Ejercicio 7
===========

Número Factorial 
----------------

Descripción:
Lee un número entero mayor a 0 y calcula su factorial de ese número.

.. important::
    Recuerda que el número factorial es el producto de todos los números enteros positivos menores o iguales a ese número. Un ejemplo es el 5!, el cual sería 5 * 4 * 3 * 2 * 1 = 120. Nota importante: el factorial de 0 es 1.  

.. code-block:: console

   Input: 5
   Output: 120

   Input: 0
   Output: 1
