Ejercicio 1
===========

Contar vocales
--------------

Descripción:
Lee una cadena y cuenta cuántas vocales ``(a, e, i, o, u)`` contiene. Ignora mayúsculas.

Ejemplo de input y output:

.. code-block:: console

   Input: Hola Mundo
   Output: 4

   Input: Hola Como Estas
   Output: 6
