Ejercicio 8
===========

Inversión de Cadena de texto
----------------

Descripción: Lee una cadena de texto e invierte la palabra. Ignora las mayúsculas.

Ejemplo de input y output:

.. code-block:: console

   Input: pepe
   Output: epep

   Input: amor
   Output: roma
